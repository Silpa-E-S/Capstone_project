Project Title :

Predicting income levels using U.S Census data : A Machine Learning Approach

Aim of the Project :

To develop a machine learning model that predicts whether an individual earns more than $50K per year using demographic and

employment-related attributes from the U.S. Census dataset.

Data Description :

The dataset ued in this project is the Adult Income Dataset from the UCI Machine Learning Repository.

It contains information collected from the 1994 U.S. Census database.The goal is to predict whether a person earns more than

$50K per year based on their personal and professional attributes.

Total Instances (Rows) :48,842

Total Features (columns) :14

Some key attributes includes : age,education,occupation,hours-per-week,native-country,income.

The dataset includes both categorical and numerical variables.

Missing values are represented by ? and appear in columns like workclass,occupation and native-country.
